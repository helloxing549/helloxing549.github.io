Minecraft(基岩版)命令简介

发表于 2020-04-22

阅读须知

首先，你要知道用 < > 和 [ ] 的区别，用 < > 括起来的内容是必填项，用 [ ] 括起来的内容为选填项中间用空格隔开, | 符号是 或 的意思。这是所有游戏命令通用的！！！
例如 /give <玩家> <物品> [数量] [数据值] [JSON数据] - 其中的”实体”和”物品”就是必填项

其实你可以去维基百科看这些命令怎么用，直接在右上角搜索你想要了解的命令就好了。那里写的比我详细！
>点我前往我的世界维基百科<>

/gamemode

语法： /gamemode <游戏模式> [玩家]
更改某位玩家的游戏模式。
游戏模式可以用 0(生存模式) 1(创造模式) 2(冒险模式) c(创造模式) s(生存模式) a(冒险模式) d(默认游戏模式)
[玩家]参数不填默认指命令执行者

/give

语法：/give <玩家> <物品> [数量] [数据值] [JSON数据]
给某个玩家物品
常见的物品有 quartz_block(石英块) bedrock(基岩) stick(木棍) appleenchanted(附魔苹果)
[数量]参数不填默认为1，最大为64。数据值默认为0，这个可以决定羊毛混凝土的颜色等。[JSON数据]这里不讲，这个可以给予能在冒险模式放置的方块或能破坏破坏指定方块的工具等。感兴趣的话可以上网查。

/xp

语法：/xp <经验值> [玩家] 或 /xp <经验值>L [玩家]
给予玩家等级
则个<经验值>如果后面有L则是给予等级数而不是经验值。

/scoreboard

/scoreboard objectives

/scoreboard objectives add <计分项> dummy [计分板名称] - 添加一个计分项
/scoreboard objectives remove <计分项> - 移除一个计分项
/scoreboard objectives list - 列出当前的所有计分板
/scoreboard objectives setdisplay <sidebar|list> [计分项] [ascending|descending] - 设置某个计分项的显示位置，如果[计分项]参数没有填写则清空这个位置的显示区域
/scoreboard objectives setdisplay <belowname> <计分项> - 设置某个计分项的显示位置为玩家名字下方

/scoreboard players

/scoreboard players list <玩家> - 列出某个玩家的全部计分项的分数
/scoreboard players reset <玩家> [计分项] - 重置某个玩家的某个计分项分数，如果[计分项]参数没有填写则重置该玩家所有的分数
/scoreboard players test <玩家> <计分项> <最小分值> [最大分值] - 检测某玩家的某个计分项分数，如果[最大分值]没有填写则为2147483647
/scoreboard players random <玩家> <计分项> <最小值> <最大值> - 将某玩家的分数设置为<最小值> <最大值>之间(包含)
/scoreboard players <set|add|remove> <玩家> <计分项> <分数> - 设置，添加，减少某玩家的某计分项分数
/scoreboard players operation <玩家> <计分项> <操作符> <玩家> <计分项> - 将某个玩家的某个计分项分数与另一个玩家的某个计分项分数进行操作运算
其中的<玩家>可以是某个生物，甚至是一个不存在的名字.
操作符又以下几个: =, +=, -=, =, /=, >, <, ><.这里不做过多的介绍
单个实体的分数范围为(-2147483648 - 2147483647)的整数
其中<玩家>参数可用代替所有实体(包括不存在的玩家和下线的玩家等),分数数值也可用*代表最小值或最大值

/clear

语法：/clesr [玩家] [物品] [数据值] [最大数量]
删除某玩家的某物品
如果所有参数都不填写则为清楚命令执行者的全部物品。
如果填写了[物品]参数则只删除填写的物品。
如果[最大数量]参数为0则为检测某晚间是否具有某物品

/setblock

语法：/setblock <坐标: x y z> <方块> [数据值] [replacr|destory|keep]
设置某坐标的方块
replace - 替换该坐标方块
destory - 将该坐标的方块破坏并设置该坐标方块
keep - 将该坐标的空气设置成某方块

/fill

语法：/fill <坐标1: x y z> <坐标2: x y z> <方块1> [destory|hollow|keep|outline|replace] [方块2] [数据值]
填充两个坐标之间的方块
destory - 将两坐标之间的方块破坏并填充两坐标之间的方块
hollow - 填充两坐标之间的轮廓，并将内部填充为空气
keep - 只填充空气
outline - 填充两坐标之间的轮廓，内部不受影响
replace - 替换方块2为某方块1

/clone

语法：/clone <起始坐标: x y z> <终止坐标: x y z> <目标坐标: x y z> [filtered|masked|replace] [force|move|normal]
将起始坐标到终止坐标之间的方块克隆到目标坐标
这里不细讲，注意选定区域的基准点，不会用尽量别用
